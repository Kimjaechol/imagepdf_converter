PDF 변환기 사용 가이드

1단계: 파일 선택

  • 드래그 앤 드롭: 파일 탐색기에서 PDF를 앱 화면으로 끌어다 놓으세요
  • 클릭 선택: 드롭 영역을 클릭하면 파일 선택 창이 열립니다
  • 폴더 일괄: "폴더 일괄 변환" 모드를 선택하면 폴더 내 모든 PDF를 한번에 변환합니다

2단계: 저장 위치

"찾아보기" 버튼을 눌러 변환 결과를 저장할 폴더를 선택하세요.

3단계: 변환 시작

"변환 시작" 버튼을 누르면 자동으로 변환이 시작됩니다. 진행률을 오른쪽 "작업 목록"에서 실시간 확인 가능합니다.

설정 안내

  • OCR 엔진: Surya가 한국어 인식에 가장 정확합니다
  • 읽기 순서 AI: 다단, 표가 많으면 "하이브리드" 추천
  • 오탈자 교정: 한국어 법률/시험 문서에 최적화

처리 특징

  • PDF를 10페이지씩 잘라서 병렬 처리 (빠른 속도)
  • 이미지 PDF(스캔문서)와 디지털 PDF 모두 지원
  • 표, 다단, 각주, 풍선글, 그래프 등 복잡한 레이아웃 인식
  • 甲乙丙丁, ⅠⅡⅢ, 법률용어 등 한국어 특화 교정
