Features

왜 MoA 문서 변환기인가?

단순 텍스트 추출이 아닌, AI가 문서의 구조와 맥락을 이해합니다.

🧠

AI 레이아웃 분석

Surya AI 기반으로 제목, 본문, 표, 그림, 수식, 각주, 다단 구조까지 자동으로 분석합니다.

🔍

고정밀 OCR

한국어, 영어, 중국어, 일본어 지원. 스캔 PDF도 원본과 동일한 텍스트로 복원합니다.

📊

표 구조 인식

셀 병합, 다페이지 연결 표, 투명 테두리 표까지 정확하게 HTML 테이블로 변환합니다.

📖

읽기 순서 AI

규칙 기반 + VLM(Gemini) 하이브리드로 다단 레이아웃도 자연스러운 순서로 정렬합니다.

✨

자동 오류 수정

한자, 로마숫자, 법률 용어 등 OCR 오류를 사전 + LLM으로 자동 보정합니다.

⚡

병렬 처리

PDF를 10페이지 단위로 분할하여 동시 처리. 수백 페이지도 빠르게 변환합니다.

🌐

번역 기능

7개 언어 간 자동 번역 지원. 문서 변환과 동시에 번역까지 한 번에 완료합니다.

📁

폴더 일괄 변환

폴더 전체 문서를 일괄 처리. 대량 문서 마이그레이션에 최적입니다.

How It Works

3단계로 완벽한 변환

복잡한 문서도 AI 파이프라인이 자동으로 처리합니다.

01

문서 업로드

PDF, DOCX, HWPX, XLSX, PPTX 파일을 드래그 앤 드롭하거나 폴더를 선택합니다.

📂
02

AI 분석 & 변환

레이아웃 감지 → OCR → 표 인식 → 읽기 순서 → 제목 분류 → 오류 수정 파이프라인이 자동 실행됩니다.

🤖
03

HTML/Markdown 출력

원본 레이아웃이 보존된 깨끗한 HTML과 Markdown 파일을 받아보세요. 이미지와 스타일시트도 함께 포함됩니다.

✅
